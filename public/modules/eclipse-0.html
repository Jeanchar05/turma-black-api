ESTRATÉGIA DE ESPERA

🌑 Eclipse 0

O Eclipse 0 trabalha com repetição direta, pausa de 4 giros e entrada somente se nenhum número do Terminal 0 com 2 vizinhos aparecer durante a espera.

REGRA PRINCIPAL

Como funciona?

Quando um número repete de forma direta, exemplo 12, 12, a leitura é ativada. Depois disso, você aguarda exatamente 4 giros.

Durante esses 4 giros, não pode aparecer nenhum número próximo do Terminal 0 com 2 vizinhos. Se aparecer, a leitura é cancelada.

3

4

Quando cancela?

Se durante os 4 giros aparecer qualquer número do Terminal 0 com 2 vizinhos, a leitura é invalidada e você não entra.

Proteção

Se a entrada não bater de primeira, pode fazer apenas mais uma proteção. Se não vier novamente, encerre a estratégia.

Quando voltar?

Depois de invalidar ou não bater, espere uma nova repetição direta para iniciar outra leitura do Eclipse 0.

Nenhuma estratégia garante resultado. Use com gestão e responsabilidade.
